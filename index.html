4-Week Plan

Learn pencil grips, pressure, and basic mark-making exercises.

Lines & Exercises
Practice straight lines, curved lines, ghosted lines, and ellipses.

Basic Shapes
Draw boxes, circles, triangles, and cylinders from observation and imagination.
Week 4
Combining Shapes
Break down objects into basic shapes. Draw from reference using shape analysis.

Sketchbook Practice
Every day: Fill at least one page with line exercises before doing anything else. Try ghosted lines, ellipses, hatching, and boxes. The goal isn't perfect lines — it's building muscle memory through repetition.

Weekly: Pick one everyday object and draw it using only basic shapes. Don't worry about detail — just shapes.
